# Mono Patch - Hex Color Generator

Mono Patch is a modern, sophisticated color palette generator designed to create harmonious monochromatic color schemes instantly. Built with a premium "glassmorphism" aesthetic, it offers a seamless experience across all devices.

Features

- **Instant Palette Generation**: Generate 5 distinct monochromatic shades based on a random base hue with a single click.
- **Smart Color Harmony**: Uses HSL calculations to ensure consistent saturation and balanced lightness distribution (20% to 80%).
- **One-Click Copy**: Click any hex box to instantly copy the color code to your clipboard.
- **Premium UI**: Features a frosted glassmorphism design with an animated mesh gradient background.
- **Dark Mode Support**: Automatically syncs with your system theme (Light/Dark) and includes a manual toggle for preference override.
- **Responsive Design**: Optimized layout for both desktop and mobile, ensuring a perfect view everywhere.

Technologies

- **HTML5**: Semantic structure.
- **CSS3**: Advanced styling with Variables, Flexbox, Animations, and Backdrop Filter.
- **JavaScript (ES6+)**: Logic for color math (HSL to Hex), DOM manipulation, and theme management.

How to Use

1. Open `ive-link` in your browser.
2. Click **Generate Palette** to create a new set of colors.
3. Click the **Copy** icon on any color strip to save the Hex code.
4. Toggle the **Sun/Moon** icon in the header to switch themes.
